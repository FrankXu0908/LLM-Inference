# FX Tools

These scripts are generic helpers for FX graph extraction, visualization, and operator mapping.

## Subfolders

- `fx_graph/`
FX extraction and graph visualization utilities.

- `operator_mapping/`
Operator mapping and operator statistics utilities.

## Model Outputs

Write model-specific outputs under:

- `fx/models/qwen3_5_35b_a3b/`
- `fx/models/qwen3_8b_dense/`

Avoid writing new graph artifacts directly under `fx/`.

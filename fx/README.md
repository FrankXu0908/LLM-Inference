# FX Track

This folder contains model graph / operator mapping analysis.

## Structure

- `fx/analysis/fx_graph/`
  - FX extraction and visualization scripts
- `fx/analysis/operator_mapping/`
  - Operator mapping/statistics scripts
- `fx/docs/`
  - FX/model-operator focused documentation
- `fx/artifacts/`
  - exported model graph artifacts
- `fx/fx_graphs/`
  - layer-level exported graph files

## Config

FX scripts use `configs/model.yaml` as the default model config path.

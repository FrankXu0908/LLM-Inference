# Qwen3.5-35B-A3B FX Graphs

This folder contains the FX/operator artifacts for the older Qwen3.5-A3B analysis track.

## Contents

- `config.yaml`
Model-specific FX metadata and paths.

- `fx_graphs/`
Layer-level exported graph text files.

- `artifacts/`
Raw model graph and layer listing artifacts.

- `docs/`
Model/operator notes for this model.

## Role

This is supporting material for the secondary Qwen3.5-A3B case study. It is not the primary optimization target anymore, but it remains useful for understanding operator structure, attention variants, and possible fusion candidates.
